# cfp-test1

GitHub 連携ありで Cloudflare Pages の練習。

"Assets only"(スタティック)なコンテンツのみ。
[heiwa4126/cfp-test0](https://github.com/heiwa4126/cfp-test0)からフォークした。

## 手順

```sh
pnpm create cloudflare@latest cfp-test1
# "Hello World example" を選ぶ
# "Assets only" を選ぶ
# Gitはお好みで
cd cfp-test1
pnpm dev  # ローカルでテスト
```
